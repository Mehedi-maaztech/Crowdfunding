# Crowdcube-Flan

 Crowdcube-Flan is a modern crowdfunding platform that connects project creators with supporters. It allows users to create, manage, and contribute to fundraising campaigns easily, offering a transparent and user-friendly interface for tracking progress, viewing campaign details, and supporting meaningful ideas.

## Live Demo

[Visit the live site](https://crowdcube-flan.netlify.app/)

## Features

* Clean, responsive UI built to work on desktop and mobile
* Various routes / pages (e.g., `/users`, `/campaigns`, etc) to fetch and display data
